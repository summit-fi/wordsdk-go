# wordsdk-go

Go SDK and command-line interface for the Word translation service.

The Word translation service provides a simple API to manage translations for your application, with support for the [Fluent](https://projectfluent.org/) localization system. 

This SDK allows you to easily fetch translations from the Word service and use them in your Go applications.
 
 - How to use the CLI to export translations -> [cli](./cli.md)
 - How to use the SDK in your Go project -> [wordsdk-go](./wordsdk.md)
 - How to use the DateTime function in Fluent -> [fluent/datetime.md](./fluent/datetime.md)
 - How to use the Number & Currency functions in Fluent -> [fluent/number.md](./fluent/number.md)
